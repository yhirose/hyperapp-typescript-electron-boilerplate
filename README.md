# hyperapp-typescript-electron-boilerplate

Hyperapp + TypeScript + Electron boilerplate

 * No module bundler is used such as webpack
 * Auto reload
 * Redux DevTools
 * Jest test framework

## Getting started

```bash
# install
git clone https://github.com/yhirose/hyperapp-typescript-electron
npm install

# setup watch build
npm watch

# start the app
npm start
```
